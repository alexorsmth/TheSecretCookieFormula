Tech stack:

- Frontend: React, TypeScript, Material UI
- Backend: Ruby on Rails
- Database: SQLite
- Styling: CSS, custom font, MUI components

General Features:

- Basic soft cookie recipe

- Ingredient slider with fixed ranges (nothing too crazy)

- Rails backend analysis for outcome on cookies

- Saved recipes and dupe recipe validation
  
- deletion of recipes
  
- Receipt/diner styled ui
